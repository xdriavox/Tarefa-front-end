HTML

HTML (HyperText Markup Language) é a linguagem de marcação utilizada para criar a estrutura de páginas web. Ele define os elementos da página, como cabeçalhos, parágrafos e links.

CSS

CSS (Cascading Style Sheets) é a linguagem usada para descrever a apresentação de um documento HTML. Com CSS, você pode controlar o layout, cores e fontes dos elementos da página.

JavaScript

JavaScript é uma linguagem de programação que permite criar conteúdos interativos em páginas web. Ele pode manipular o DOM (Document Object Model) para atualizar o conteúdo e responder a eventos.
